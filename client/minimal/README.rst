=====================
MEMO vite development
=====================

Local install::

  # in folder ./client/minimal/
  $ npm install

Start development server::

  $ ./manage vite.minimal.dev

  # in folder ./client/minimal/
  $ npm exec -- vite

Fix source code::

  # in folder ./client/minimal/
  $ npm run fix

Fix & Build::

  $ ./manage vite.minimal.build
